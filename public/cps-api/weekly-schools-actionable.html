Weekly Schools Actionable

This page queries the CPS API for all the weeks in which each school has reported “actionable cases” of COVID. It includes the latest case and contact counts, the most recent week in which a case was reported, the number of weeks in which cases have been reported for SY2021-22 and sums all the cases and contacts reported by CPS this year. The table is sorted alphabetically by school name, just as the CPS output is. To resort, use the Copy to Clipboard button and paste the data into a separate spreadsheet.
